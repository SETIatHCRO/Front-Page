Power Cable Notes:

RFCB power Cables

	LO1   x4      20”   BLU  BLK   YLW

	LO2   x2     14”    BLU  BLK   YLW  WHT

	PD+AMP x2    12”    RED  BLK

	Down 4    x8    9.5”    YLW  BLK

Use 22 Ga. Stranded Teflon insulated wire twist to ~ 2turns per inch.

Cut length tol. +2/-0 inches.
